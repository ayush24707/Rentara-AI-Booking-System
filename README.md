# Rentara AI Booking System

AI-powered booking platform designed to improve booking experiences through dynamic pricing, chatbot assistance, and real-time availability management.

## Features

- Real-Time Availability Tracking
- Dynamic Pricing System
- AI Chatbot Assistance
- Booking Management Dashboard
- User Authentication
- Responsive UI

## Tech Stack

### Frontend
- React.js

### Backend
- Django
- Node.js

### Database
- MongoDB
- PostgreSQL

### AI/ML
- Machine Learning
- NLP Chatbot
